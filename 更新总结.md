# 更新总结 - 2026-03-12

## 本次更新内容

### 1. 修复 QThread 崩溃问题 ✅

**问题**：点击 Tag 模式切换时，应用关闭后出现 `QThread: Destroyed while thread is still running` 崩溃。

**根本原因**：
- `RemoteTagLoader` 线程在对话框关闭时未正确等待线程结束
- 主窗口的 `closeEvent` 方法缺少对 `tag_loader` 线程的清理

**修复方案**：
1. 在 `_on_tags_loaded` 方法中添加 `self.tag_loader.wait()` 确保线程正常结束
2. 在 `closeEvent` 方法中添加 `tag_loader` 线程清理逻辑

**修改文件**：
- `src/views/main_window.py:590-619`（closeEvent 方法）
- `src/views/main_window.py:1351-1365`（_on_tags_loaded 方法）

### 3. 现代化 LoadingWidget ✅

**问题**：原有的三点跳动 loading 动画视觉单调，不够现代化。

**解决方案**：设计了4种现代化 loading 样式。

#### 可用样式

1. **STYLE_SPINNER（旋转圆环）** - 推荐默认
   - 流畅60fps动画
   - 蓝→紫→粉渐变色
   - 适合通用场景

2. **STYLE_DOTS（跳动圆点）**
   - 三色圆点上下跳动
   - 简洁轻快
   - 适合快速加载

3. **STYLE_PULSE（脉冲圆环）**
   - 圆环缩放呼吸
   - 半透明外圈
   - 适合持续任务

4. **STYLE_BARS（竖条动画）**
   - 五彩竖条波浪
   - 音乐可视化风格
   - 适合创意场景

#### 技术特性
- ✅ QPainter 硬件加速渲染
- ✅ 抗锯齿圆滑效果
- ✅ 自动资源清理
- ✅ 完全向后兼容

#### 使用方法

**基础用法（默认样式）**：
```python
from src.widgets.loading_widget import LoadingWidget

loading = LoadingWidget("加载中...")  # 默认 STYLE_SPINNER
loading.start_animation()
```

**指定样式**：
```python
loading = LoadingWidget("加载中...", LoadingWidget.STYLE_DOTS)
loading.start_animation()
```

**动态切换**：
```python
loading.set_style(LoadingWidget.STYLE_PULSE)
loading.set_text("处理中...")
```

**独立对话框**：
```python
from src.widgets.loading_widget import ModernLoadingDialog

dialog = ModernLoadingDialog("加载中...", LoadingWidget.STYLE_SPINNER, self)
dialog.start()
# ... 执行操作 ...
dialog.stop()
```

#### 兼容性说明
- ✅ **完全向后兼容**：现有代码无需修改
- ✅ **默认优化**：自动使用最佳的旋转圆环样式
- ✅ **API 不变**：`start_animation()` 和 `stop_animation()` 保持一致
