# 现代化对话框设计方案 - 替代 QMessageBox

## 问题分析

**当前使用的 QMessageBox**：
- ❌ 原生系统对话框，视觉过时
- ❌ 与应用的 Glassmorphism 风格不搭
- ❌ 无动画效果，体验生硬
- ❌ 样式无法自定义

---

## 设计方案：Glassmorphism Modal Dialog

基于 UI/UX Pro Max 分析和 PodPilot 的设计风格，设计现代化对话框组件。

### 设计理念

**核心特点**：
1. **Glassmorphism 风格**：半透明、模糊背景、微光边框
2. **平滑动画**：淡入淡出 + 缩放效果（200-300ms）
3. **层次分明**：图标 + 标题 + 消息 + 按钮
4. **状态色彩**：成功/警告/错误/信息用不同颜色

## 技术实现

### 组件结构

```python
class ModernDialog(QDialog):
    """现代化对话框基类"""
    
    TYPE_INFO = "info"
    TYPE_SUCCESS = "success"
    TYPE_WARNING = "warning"
    TYPE_ERROR = "error"
    TYPE_QUESTION = "question"
    
    def __init__(self, title, message, dialog_type, buttons, parent=None):
        super().__init__(parent)
        # 无边框、透明背景
        self.setWindowFlags(Qt.FramelessWindowHint | Qt.Dialog)
        self.setAttribute(Qt.WA_TranslucentBackground)
        
        # 淡入动画
        self.fade_in_animation()
```

### 核心特性

#### 1. Glassmorphism 样式
```python
background: qlineargradient(
    x1:0, y1:0, x2:0, y2:1,
    stop:0 rgba(30, 30, 46, 0.95),
    stop:1 rgba(20, 20, 35, 0.95)
);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.12);
border-radius: 16px;
box-shadow: 0 8px 32px rgba(0, 0, 0, 0.4);
```

#### 2. 图标设计
```python
# 圆形背景 + SVG 图标
icon_container = QFrame()
icon_container.setFixedSize(60, 60)
icon_container.setStyleSheet(f"""
    QFrame {{
        background: qlineargradient(
            x1:0, y1:0, x2:0, y2:1,
            stop:0 {color_light},
            stop:1 {color_dark}

    }}
""")
```

**图标映射**：
- Info: `ℹ️` 或 info SVG icon
- Success: `✓` 或 check SVG icon
- Warning: `⚠` 或 alert-triangle SVG icon
- Error: `✕` 或 x-circle SVG icon
- Question: `?` 或 help-circle SVG icon

#### 3. 动画效果
```python
# 淡入动画（200ms）
opacity_effect = QGraphicsOpacityEffect()
self.setGraphicsEffect(opacity_effect)
opacity_animation = QPropertyAnimation(opacity_effect, b"opacity")
opacity_animation.setDuration(200)
opacity_animation.setStartValue(0)
opacity_animation.setEndValue(1)
opacity_animation.setEasingCurve(QEasingCurve.OutCubic)
opacity_animation.start()

# 缩放动画（200ms）
self.scale_animation = QPropertyAnimation(self, b"geometry")
```

## 使用方式

### 简化 API

```python
# 信息提示
ModernDialog.information(self, "提示", "当前项目中没有发现使用tag引用的Pod")

# 成功提示
ModernDialog.success(self, "成功", "切换完成：成功 3 个，失败 0 个")

# 警告
ModernDialog.warning(self, "警告", "以下Pod未选择Tag:\n• PodA\n• PodB")

# 错误
ModernDialog.error(self, "错误", "处理MR信息时发生错误: 网络连接超时")

# 确认对话框
reply = ModernDialog.question(
    self, 
    "确认", 
    "确定要为 3 个Pod切换Tag吗？\n\n这将修改Podfile中的Pod引用。"
)
if reply == ModernDialog.Yes:
    # 用户点击确定
    pass
```

## UX 最佳实践

### DO ✅
1. **明确的视觉层次**：图标 → 标题 → 消息 → 按钮
2. **状态色彩区分**：不同类型用不同颜色
3. **平滑动画**：200-300ms 过渡
4. **清晰的按钮层次**：主按钮高亮，次按钮低调
5. **适当的留白**：24px padding，16-24px 元素间距

### DON'T ❌
1. **过度动画**：避免超过 300ms 的动画
2. **emoji 图标**：使用 SVG 或 Unicode 符号
3. **多个主按钮**：最多一个高亮按钮
4. **过长文本**：超过 3 行考虑滚动
