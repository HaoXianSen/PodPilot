## 修复的问题

### 问题1：复选框不显示 ✅

**原因**：SVG 图标路径错误或加载失败

**解决**：使用原生复选框样式 + CSS 自定义
```css
QCheckBox::indicator {
    width: 18px;
    height: 18px;
    border: 2px solid rgba(255, 255, 255, 0.3);
    border-radius: 4px;
    background: rgba(255, 255, 255, 0.05);
}
QCheckBox::indicator:checked {
    background: #3b82f6;
    border-color: #3b82f6;
}
```

### 问题2：内容被压缩 ✅

**原因**：content_layout 没有弹性空间，内容被底部按钮压缩

**解决**：
```python
self.content_layout.addWidget(options_container)
self.content_layout.addStretch()  # ← 添加弹性空间
```

### 问题3：警告卡片过于复杂 ✅

**原因**：红色卡片背景在深色主题下过于抢眼

**解决**：改为简洁的文案提示
```python
warning_text = QLabel("⚠️ 此操作不可逆，请谨慎选择")
warning_text.setStyleSheet("""
    color: #fbbf24;  # 黄色警告色
    font-size: 12px;
    padding: 8px 0;
""")
```

## 技术细节

### 嵌套布局
每个选项使用独立的 QWidget 包装，避免布局压缩：
```python
pods_option = QWidget()
pods_option_layout = QVBoxLayout(pods_option)
├── QCheckBox("删除 Pods 目录")
└── QLabel("移除所有已安装的 Pod 库文件")
```

### 复选框状态管理
```python
self.clean_pods_cb.stateChanged.connect(
    lambda state: self._update_option("pods", state == Qt.Checked)
)

def _update_option(self, option, checked):
    self.clean_options[option] = checked
```

### 数据传递
```python
# main_window.py
dialog = CleanCacheDialog(self)
if dialog.exec_() == QDialog.Accepted:
    options = dialog.get_options()
    clean_pods = options.get("pods", False)
    clean_lock = options.get("lock", False)
    clean_cache = options.get("cache", False)
```
