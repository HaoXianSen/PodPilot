# PodPilot UI 现代化重构设计文档

**日期:** 2026-03-10
**状态:** 已确认

## 1. 概述

### 1.1 目标
将 PodPilot 主界面从传统 PyQt5 样式升级为现代 Glassmorphism 风格。

### 1.2 当前问题
- 使用 emoji 作为图标，跨平台显示不一致
- 样式分散，缺乏现代感
- 视觉层次不够清晰

## 2. 设计决策

| 决策项 | 选择 | 原因 |
|--------|------|------|
| 视觉风格 | Glassmorphism | 现代感强，毛玻璃效果时尚 |
| 图标方案 | SVG 矢量图标 | 完美缩放，可定制颜色 |
| 主题支持 | 仅暗色 | 简化开发，专注单一风格优化 |

## 6. 技术要点

### 6.1 PyQt5 Glassmorphism 实现
由于 PyQt5 不直接支持 CSS `backdrop-filter`，采用以下方案：
- 使用半透明背景色 `rgba(255, 255, 255, 0.15)`
- 添加细边框增强层次感
- 圆角 12px 现代感设计

### 6.2 SVG 图标
使用 Lucide 图标风格，内联 SVG 路径：
- 支持动态颜色
- 支持任意尺寸缩放
- 缓存机制提升性能
